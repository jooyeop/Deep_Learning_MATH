## 최적화 문제
- 극값 찾기 : 최대/최소값, 정류점(Stationary Points)
- Lagrange Multiplier : 제약이 있는 최적화 문제
- Gradient Descent 알고리즘 : 학습 과정에서 손실 함수 최소화를 위한 방법론, 경사하강법과 연결

## 곡선과 면의 기하학
- 곡선의 길이, 면적
- 다변수 함수의 그래프와 접선 및 접면
- Hessian 행렬 : 2차 미분을 이용한 함수의 곡률과 최적화 문제에서의 응용
